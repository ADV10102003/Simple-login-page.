# Simple Login System (HTML + CSS + JS)

This project contains a clean and modern login page with a display page that shows the logged-in username.

## Project Files
- index.html – Login Page  
- display.html – Show username  
- style.css – UI and background  
- script.js – Login logic  
- README.md – Documentation  

## How to Use
1. Open index.html  
2. Enter username + password  
3. Redirects to display.html and shows username
